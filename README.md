## 🚀 Usage

**Get the Source Code**

```shell
$ git clone https://github.com/anyesu/taro-demo
$ cd taro-demo
```

**Installation**

```shell
$ pnpm i
```

**Run**

```shell
$ # cd packages/taro-demo-react
$ cd packages/taro-demo-vue3

$ pnpm dev:h5
```
